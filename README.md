<h3>为什么要学习本课程</h3>
<p>我们先来讨论一个问题为什么要学习Vue?</p>
<p>Angular国内使用率低</p>
<p>相对而言，Angular在国内的使用率是较低的，当然Angular本身是非常优秀的框架。</p>
<p>React国内使用率高</p>
<p>而React是在国内外使用率都非常高的，特别是国内的很多大公司在之前，都会更加倾向于一些项目选择React，这是因为在之前React有更高的灵活度，也包括更好的对TypeScript的支持。3.Vue国内使用率最高但是在国内长久以来，使用率最高的是Vue,在国内几乎所有的前端岗位都会对Vue有要求，包括大量的项目都是使用Vue来完成的。甚至在国内我们可以得出这样的结论:只要学好了Vue你一定可以找到一份前端的工作，如果你没有掌握Vue,很难找到一份前端的工作。虽然这样的说法会有一些绝对，但是在找工作的过程中，确实会有这样的感受，也体现了Vue对于前端开发者的重要性所以，对于前端工程师来说，Vue框架是必须熟练、深入掌握的一个框架。</p>
<p>Vue3的发布以及带来新的特性</p>
<p>在2020年的9月19日，万众期待的Vue3终于发布了正式版，命名为“One Piece” 。并且它也带来了一大波新的特性:更好的性能、更小的包体积、更好的TypeScript集 成、更优秀的APl设计。在Vue3发布之初，很多人也是跃跃欲试，想要尝试Vue3的各种新特性。但是事实上在刚刚发布的时候我们使用Vue3来写demo练习是没有问题的，真正在实际业务项目中使用Vue3还需要一个相对的过程:包括Vue3的进一步 稳定、包括社区更多Vue3相关的插件、组件库的支持和完善。</p>
<p>那么现在是否是学习Vue3的时间呢?</p>
<p>答案是肯定的，首先Vue3在经过一系列的更新和维护后，已经是趋于稳定，并且在之前尤雨溪也宣布在今年(2021年)第二季度会将Vue3作为Vue CLI的默认版本了。目前社区也经过- -定时间的沉淀，更加的完善了，包括AntDesignVue、 Element-Plus都提供了对Vue3的支持，所以很多公司目前新的项目都已经在使用Vue3来进行开发了。</p>
